# Baymax

Baymax is an AI-vs-AI Chess Arena built with FastAPI, LangGraph, LangChain, Stockfish, python-chess, Ollama, and React.

The project is designed to run fully locally using free and open-source tools first, while keeping the architecture deploy-ready for future paid APIs and cloud services.

## Core Idea

- Stockfish handles strong chess move calculation.
- python-chess handles board rules and legal move validation.
- LangGraph controls the AI agent workflow.
- Ollama provides local open-source LLM support.
- FastAPI provides the backend API.
- React provides the chess UI.
